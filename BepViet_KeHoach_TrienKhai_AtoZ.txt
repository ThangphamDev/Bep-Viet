
BẾP VIỆT – KẾ HOẠCH TRIỂN KHAI TỪ A→Z (BACKEND NODE.JS + FLUTTER DART)
==============================================================================
MỤC TIÊU
- Ứng dụng giải quyết “Hôm nay ăn gì?” với gợi ý theo vùng miền, mùa vụ, ngân sách.
- Lập kế hoạch tuần, tạo danh sách đi chợ, quản lý “Tủ lạnh của tôi”.
- Cộng đồng công thức, gói Premium với hồ sơ gia đình và cảnh báo dinh dưỡng.
- Backend Node.js (khuyến nghị NestJS + TypeScript). Mobile FE Flutter (Dart). DB MySQL.

LỘ TRÌNH & TASKS (ROADMAP)
------------------------------------------------------------------------------
PHASE 0 – Khởi tạo & CI/CD (1 tuần)
- [DevOps] Quy ước repo, branch, commit (Conventional Commits), code style (ESLint, Prettier).
- [Infra] Docker Compose: mysql, adminer, api, flutter web (dev).
- [CI] GitHub Actions: Lint + Test + Migrate DB + Build Docker.
- [Security] .env mẫu & vault, git-secrets, renovate/bot.
Deliverables: /docker, /infra, pipeline CI; README khởi tạo dự án.

PHASE 1 – Nền tảng dữ liệu & Auth (2 tuần)
- [DB] Áp dụng schema MySQL đã cung cấp (22+ bảng core + mở rộng).
- [Auth] Đăng ký/Đăng nhập (email + password), JWT, refresh token, RBAC (user, admin).
- [User Profile] Thêm region/subregion, khẩu vị, hạn chế dinh dưỡng.
Deliverables: Migrations, Seed cơ bản; /auth, /users module; Postman collection.

PHASE 2 – Công thức & Gợi ý theo vùng/mùa/ngân sách (3 tuần)
- [Recipes] CRUD recipes, variants theo Bắc/Trung/Nam; bước nấu, ảnh.
- [Seasonality] Seasons + ingredient_seasonality; ưu tiên gợi ý theo mùa.
- [Budget] ingredient_prices theo vùng; ước tính chi phí món và bữa.
- [Suggest API] Gợi ý theo (region, season, servings, budget).
Deliverables: /recipes, /variants, /ingredients, /prices, /suggestions.

PHASE 3 – Kế hoạch tuần + Danh sách đi chợ + Tủ lạnh (3 tuần)
- [Planner] Tạo meal_plan (7 ngày), tránh lặp, cân bằng chi phí.
- [Shopping] Tổng hợp nguyên liệu từ kế hoạch → shopping_list + group theo quầy.
- [Pantry] Quản lý tồn kho, cảnh báo HSD, trừ tồn khi nấu.
Deliverables: /meal-plans, /shopping-lists, /pantry.

PHASE 4 – Premium (Family Profile) & Cảnh báo (2 tuần)
- [Premium] Subscriptions, family_profiles/members, allergies, hạn chế.
- [Advisory] Cảnh báo cay/dị ứng/huyết áp/tiểu đường theo hồ sơ.
Deliverables: /subscriptions, /family, /advisory endpoints.

PHASE 5 – Cộng đồng & Moderation (2 tuần)
- [Community] Đăng công thức, bình luận, đánh giá, like/favorite.
- [Admin] Duyệt, gắn Featured, phân tích cơ bản.
Deliverables: /community, /comments, /ratings, /moderation, /analytics.

PHASE 6 – Polish & Growth (liên tục)
- i18n, Performance, Offline, Push Notification, A/B test, Deep links.
- Tích hợp siêu thị (giá theo chợ/siêu thị, giao hàng), Video hướng dẫn.

V. LOGIC GỢI Ý (SERVICE PSEUDO)
------------------------------------------------------------------------------------
Input: region, season, servings, budget, pantry_ids[], exclude_allergens[], spice_pref, max_time
for each recipe:
  v = variant(region)
  if v has allergens in exclude_allergens -> skip
  time_penalty = (recipe.timeMin > max_time ? -20 : 0)
  est = sum( price(region, ing) * qty(servings) ) - pantry_credit(ing in pantry)
  season_score = avg( availability(ing, season) ) or default 60
  region_bonus = (recipe.base_region == region ? +10 : 0)
  budget_bonus = (est <= budget ? +15 : - (est - budget)/5000 )
  spice_penalty = abs(recipe.spice_level - spice_pref) * -2
  score = season_score + region_bonus + budget_bonus + spice_penalty + time_penalty
sort desc score, return top N with breakdown items.

/* src/services/suggestions.service.js (ý chính) */
// req.body: { region, season, servings, budget, pantry_ids, exclude_allergens, spice_pref=1..5, max_time }
const { getRecipesWithVariant, estimateCost, seasonalScore } = require('../models/recipes.model');
exports.search = async (req, res, next) => {
  try {
    const { region, season, servings=2, budget=50000, pantry_ids=[], exclude_allergens=[], spice_pref=2, max_time=45 } = req.body || {};
    const recipes = await getRecipesWithVariant(region);
    const out = [];
    for (const r of recipes) {
      if (r.allergens?.some(a => exclude_allergens.includes(a))) continue;
      const timePenalty = r.time_min > max_time ? -20 : 0;
      const { total, items } = await estimateCost(r.variant_id, servings, region, pantry_ids);
      const sScore = await seasonalScore(r.variant_id, season);
      const regionBonus = r.base_region === region ? 10 : 0;
      const budgetBonus = total <= budget ? 15 : -Math.max(0, (total - budget))/5000;
      const spicePenalty = Math.abs((r.spice_level||0) - spice_pref) * -2;
      const score = sScore + regionBonus + budgetBonus + spicePenalty + timePenalty;
      out.push({ recipe_id: r.id, variant_region: region, total_cost: total, items, season_score: sScore, score, reason: "region/season/budget" });
    }
    out.sort((a,b)=> b.score - a.score);
    res.json(out.slice(0, 10));
  } catch (e) { next(e); }
};

/* src/routes/suggestions.js */
